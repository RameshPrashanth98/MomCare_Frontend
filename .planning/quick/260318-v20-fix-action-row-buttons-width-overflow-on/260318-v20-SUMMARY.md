# Quick Task 260318-v20: Fix Action Row Button Width

## What was done

Fixed the three action row buttons (Register Mother, Clinic Visit, Add Vaccination) on the Home screen so they fill the 393px iPhone 16 Pro container equally.

## Root cause

Each button had `whiteSpace: nowrap` and content-driven padding but no `flex: 1`, causing the three buttons to collectively overflow the action row on narrow screens. The container also used `justify-center` instead of `w-full`.

## Changes

**`app/dashboard/page.tsx`**
- Action row container: changed `justify-center` → `w-full` so it spans the full column
- Each button: added `flex: 1`, `minWidth: 0`, `justifyContent: 'center'`, and reduced `paddingLeft/Right` from `--spacing-md` to `--spacing-sm` to allow flex shrinking
- Text content preserved exactly; icons preserved

## Result

All three buttons now display as equal-width pills that collectively fill the full 393px width with consistent gap spacing — matching the wireframe.
