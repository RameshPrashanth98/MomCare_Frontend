## Wireframe Analysis

The Clinic Session screen has these sections:
1. **Header**: Back arrow, "Clinic Session" title, Search icon, "8/EN" badge, pink avatar
2. **Pink banner**: "Clinic Session" pink label, clinic name + location title, calendar date/time
3. **Clinic Info Card**: Clinic name, Active status badge, location, date/time, green Active dot, assigned midwife
4. **Registered Mothers section**: Section header
5. **Mother Cards** (repeatable):
   - Avatar circle with initials + color
   - Name, warning icon (for high-risk), NIC/area subtitle
   - Status badge (Active/Waiting)
   - Date, time range, assigned midwife, expected mothers count
   - Action buttons: Record Visit (pink outlined), View Profile (outlined), Medical History/Check In (outlined)
6. **Bottom Action Bar**: Start Clinic (pink filled), End Clinic (outlined), Add Walk-in Mother (outlined)
7. **Bottom Navigation**: HOME, MOTHERS, CLINICS (active), RECORDS, PROFILE
